# 공문서/대학 스타일 패턴 (workspace corpus 분석)

`style_pass.py` 기본값의 근거 문서. 2026-07 `~/workspace/work` 실문서 corpus에서 추출
(.hwpx 698건 발견, 137건 XML 파싱 집계 + 11건 심층 + 번들 템플릿 6종).

|------|------|------|

## 본문 스타일 — 두 regime

### Regime A — 공문/기안문/회의록/규정 (기본값, `hwp new` 기본과 일치)

- 여백 L/R/T/B/H/F = `8504/8504/5668/4252/4252/4252` = **30/30/20/15/15/15mm**
- 본문 **함초롬바탕 10pt**, 정렬 **JUSTIFY**, 줄간격 **PERCENT 160**
- 제목/헤딩 함초롬돋움, 문서 제목 **15pt 가운데** (corpus 실문서 관례)
- 번들 템플릿 6종(기안문·공문서·보고서·사업계획서·회의록) 전부 이 스타일

### Regime B — 보고서/사업계획서/외부 서식 (내용 밀도 높은 문서)

- 여백 ≈ L/R `5669`(20mm), T/B `2834`–`4251`(10–15mm) — corpus 최빈 단일 여백
- 본문 **휴먼명조 11–12pt**(실문서 최다 68건) 또는 **맑은 고딕 11–12pt**(21건, 대학/외부 서식)
- 제목 15pt(1500). 줄간격 120–160% 혼재
- style_pass 미적용 영역(폰트 교체는 렌더 폰트 가용성 리스크) — 필요 시 후속

### corpus 분포 (137건)

- 본문 폰트: 휴먼명조 68 > 함초롬바탕 30 > 맑은 고딕 21 (외 굴림체·KoPubWorld 등)
- 본문 크기: 10pt 37 / 12pt 34 / 11pt 31 / 15pt 14
- 줄간격: **160% (76)** > 130% (21) > 120% (11)

## 표 관례 (style_pass 핵심 대상)

### 세로 정렬

셀 `vertAlign="CENTER"`가 사실상 보편 (헤더·본문 공통). `TOP`은 다문단 서술형
셀에만 (KOICA 서술표, 규정 대비표).

### 헤더행

- 음영 + 가로 **CENTER** + 세로 CENTER. 굵게는 일반적(보편은 아님).
- 음영색 팔레트 (관측치):
  - 중립 회색: **`#F2F2F2`** (최빈·기본값), `#CCCCCC`, `#D6D6D6`
  - 연블루: **`#D9E2F3`**, `#D8E5F5`, `#E0E5FA` (RISE/글로컬 서식 선호)
  - 크림: `#FCF5E7`, `#F7F6F1`
  - 네이비 배너(흰 글자, 전폭 병합 제목행): `#092E99`, `#1A3072`

### 본문 셀 정렬

내용 타입 기준 — 라벨/순번/숫자/짧은 칼럼 → **CENTER**, 장문/서술 칼럼 → LEFT/JUSTIFY.

### 칼럼 폭 비율 (cellSz@width HWPUNIT 관측치)

| 패턴 | 관측 비율 | 예시 |
|------|-----------|------|
| 균등 (4col 회의록 등) | 1:1:1:1 | meetings/…/산학협력회의록_통합 |
| 2col 라벨:값 | **1:3 – 1:5** (라벨 7200–8500 = 25–30mm) | KOICA 출장보고 개요 7756:41849 |
| 2col 대비표 (현행/개정) | **1:1** | 규정 개정 대비표 23808:23910 |
| 4col 번호·구분·내용·비고 | 1 : 1.4 : 3.7 : 3.9 | KOICA T2 4082:5847:15104:15738 |
| 5col 명단 (순번·소속·성명·직위·연락처) | 1 : 4.7 : 3.7 : 2.7 : 2 | rise 참석자 명단 서식 |
| 전폭 배너 제목행 | 병합 1셀 = 전폭 + 진한 음영 | 다수 서식 T1 |

- 순번류 좁은 칼럼 최소 ≈3400 hwpu (12mm)
- 행 높이는 내용 따라 자동 (고정 높이·특수 헤더 높이 관례 없음)

## style_pass.py 기본값 (사용자 확정 2026-07-02)

- 헤더행 음영 **#F2F2F2** + 굵게 + 가운데/가운데 (`--header-fill`로 변경 가능)
- 문서 제목(H1) **가운데 15pt 굵게** (`--no-title-center` 옵트아웃)
- 칼럼 폭: 내용 표시폭 비례, 2col 라벨:값 1:3~1:4 특례, 균등 내용은 균등 유지,
  최소 3400 hwpu, 표 총폭 보존
- 짧은 칼럼(표시폭 ≤ 8) 본문 셀 CENTER
- 적용: 템플릿 없는 생성 전 경로 자동 (`--plain` 옵트아웃), 기존 파일은 `beautify`
