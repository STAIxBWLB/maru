# Integration IDs

Copy this template to the workspace-local task root, usually
`tasks/references/integration-ids.md`, and fill it with local service receipts.
Do not put real IDs in the public skill package.

## External Service IDs

| Service | Key | Value |
| --- | --- | --- |
| Google Tasks | default list | `TASK_LIST_ID` |
| Google Calendar | default calendar | `CALENDAR_ID` |
| gws CLI | binary | `GWS` |

## Active Receipts

| Task File | Google Tasks ID | Task List | Calendar | Note |
| --- | --- | --- | --- | --- |
| `tasks/active/YYMMDD-example.md` | `TASK_ID` | default | `EVENT_ID` | Example only |
