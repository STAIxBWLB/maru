# 06-proposal

Pre-selection space for this business unit. New units start proposal-first
(Tree C, naming-and-placement.md §B): categories 01~05 are activated only when
the first real obligation appears, never as empty scaffolds.

Create these canonical subdirectories on demand (do not pre-create):

- `announcement/` : RFP, guidelines, reference material from the funder
- `drafts/` : working proposal drafts
- `final/` : the submitted proposal
- `review/` : eligibility responses, interview prep, evaluator Q&A
- `contracting/` : award-to-contract paperwork in progress
