# year-cycle

Template copied into `{YYYY}-Y{N}/`.

| Stage | Purpose |
|-------|---------|
| `01-notice/` | Yearly notice, budget allocation, operating guide |
| `02-planning/` | Year plan, budget, KPI, staffing, schedule |
| `03-execution/` | Programs, spending, procurement, HR, contracts, deliverables, evidence, events, changes |
| `04-monitoring/` | Issues, risks, monthly checks, KPI tracking |
| `05-reporting/` | Interim, annual, and ad hoc reports |
| `06-evaluation/` | Self/external evaluation, Q&A, results |
| `07-settlement/` | Settlement, audit, financial evidence |
| `08-year-close/` | Handoff, retrospective, next-year bridge |
